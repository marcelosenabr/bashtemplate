## Instalação

1. Extraia os arquivos no diretório com o nome do novo script
2. Altere o nome dos arquivos `bashtemplate` e `bashtemplate.sh` com o nome do novo script
3. Executa na raiz do script `source novopacote`

```bash
source novopacote
```
